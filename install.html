Firefox
Click here.

Chrome
Paste this into the address bar. Make sure the 'javascript:' is there! Optionally change the keyword to 'k' or 'kaf'.
javascript:window.external.AddSearchProvider("http://127.0.0.1:43110/1Mr5rX9TauvaGReB4RjCaE6D37FJQaY5Ba/opensearch.xml");

Opera
Right click on the address bar and select 'Edit Search Engines...'. Click 'add new search'. Enter the following information:
Name: Kaffiene
Keyword: k
Address: http://127.0.0.1:43110/1Mr5rX9TauvaGReB4RjCaE6D37FJQaY5Ba/?q=%s
Use Post: No/unchecked.
Query String: Blank/Greyed out.
Safari
At the moment Safari does not support external search engines.
